# Análise de Dados com Pandas

Este projeto tem como objetivo realizar uma análise exploratória de dados (EDA) utilizando a biblioteca Pandas em Python.

## Sobre o dataset
O dataset utilizado contém informações sobre emendas parlamentares, incluindo dados como ano, tipo de emenda, município, autor e valores financeiros.

## Atividades realizadas
- Visualização das primeiras linhas do dataset
- Identificação da estrutura dos dados (linhas e colunas)
- Análise dos tipos de dados
- Verificação de valores nulos
- Estatísticas descritivas
- Ordenação dos dados
- Identificação de registros duplicados

## Conclusão
A análise permitiu compreender a estrutura do dataset, identificar possíveis inconsistências e preparar os dados para análises futuras.
